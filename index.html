What is Artour?

Artour is built to be easily accessible and highly navigable, with as few interruptions to navigation as possible. This allows users to stay up to date with various work events, meetups, and conventions; without the need for expensive, purpose-built solutions for single events. The in-build event creator and map tool allow for event hosts to customize menus, vendor details, and more; then publish the event so attendees can easily join via search, QR code, or join code.
